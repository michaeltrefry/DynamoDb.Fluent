### DynamoDb.Fluent

DynamoDb.Fluent is a .NET library that provides a fluent, strongly-typed API over Amazon DynamoDB, allowing callers to build queries, scans, and CRUD operations on DynamoDB tables using a chainable interface. It ships two implementations behind the same interface set: a real DynamoDB backend (using the AWS SDK's Document Model) and an in-memory backend (backed by concurrent in-memory data structures using Newtonsoft.Json JTokens), making it suitable for both production use and unit testing without a live DynamoDB connection.
