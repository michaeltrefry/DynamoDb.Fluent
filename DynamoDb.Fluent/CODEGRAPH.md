## Summary

DynamoDb.Fluent is a .NET library that provides a fluent, type-safe abstraction over AWS DynamoDB, exposing CRUD operations, query/scan builders, and index access through generic interfaces. It ships two implementations: a real DynamoDB backend (wrapping the AWS SDK's Document Model) and an in-memory backend (using JSON tokens and concurrent sorted dictionaries) intended for testing. The dual-backend design allows application code to be written against shared interfaces (IDynamoDbContext, ITable<T>, IObjectQuery<T>, etc.) and swapped between real and in-memory storage without code changes.
