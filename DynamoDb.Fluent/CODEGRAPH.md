## Summary

DynamoDb.Fluent is a .NET library that provides a fluent, strongly-typed API for interacting with AWS DynamoDB tables. It offers two interchangeable implementations: a real DynamoDB backend (via the AWS SDK) and an in-memory backend (useful for testing), both sharing the same interface hierarchy for table operations, queries, scans, and index lookups.
