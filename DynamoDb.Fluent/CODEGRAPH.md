## Summary

DynamoDb.Fluent is a .NET library that provides a fluent, strongly-typed abstraction over AWS DynamoDB, exposing table operations (put, delete, find, query) and rich query/scan condition chaining through generic interfaces. It ships two implementations: a real DynamoDB backend (via the AWS SDK's Document Model) and an in-memory backend suitable for testing, both sharing the same interface contract (IDynamoDbContext, ITable<T>, IObjectQuery<T>, etc.).
