## Summary

DynamoDb.Fluent is a .NET library that provides a fluent, type-safe abstraction layer over Amazon DynamoDB. It exposes a unified interface for querying, scanning, putting, and deleting items from DynamoDB tables, and includes a complete in-memory implementation of the same interfaces to facilitate unit testing without requiring a live DynamoDB connection.
