## Summary

DynamoDb.Fluent is a .NET library that provides a fluent, type-safe abstraction layer over AWS DynamoDB, enabling query building, CRUD operations, and index access through chainable interfaces. It includes two implementations: a real DynamoDB backend (using the AWS SDK) and an in-memory backend (using JSON tokens and concurrent sorted dictionaries) suitable for testing. The library exposes a unified interface hierarchy (IDynamoDbContext, ITable<T>, ITableIndex<T>, IObjectQuery<T>, IScanCondition<T>, IQueryCondition<T>) so consumers can swap between real and in-memory backends without changing application code.
